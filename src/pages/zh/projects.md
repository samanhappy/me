### [MCPHub](https://github.com/samanhappy/mcphub) · ⭐ 2,000+

统一管理和编排多个 MCP（Model Context Protocol）服务器的开源网关。MCPHub 将多个 MCP 服务聚合为独立端点，支持灵活的路由策略——一个地方管理所有 MCP 集成。

**技术栈：** TypeScript · Node.js · MCP 协议
**影响力：** 2,000+ GitHub Stars，300+ 开发者社区，入选 MCP 官方生态
**链接：** [GitHub](https://github.com/samanhappy/mcphub) · [文档](https://mcphub.io)

### [Selectly](https://github.com/samanhappy/selectly)

AI 驱动的浏览器文本处理扩展，支持选中任意网页文本进行翻译、解释、总结、对话——无需离开当前页面。

**技术栈：** TypeScript · Browser Extension APIs · LLM 集成
**链接：** [GitHub](https://github.com/samanhappy/selectly) · [Chrome 商店](https://chromewebstore.google.com/detail/selectly)
