### [MCPHub](https://github.com/samanhappy/mcphub) · ⭐ 2,000+

A unified hub for centrally managing and dynamically orchestrating multiple MCP (Model Context Protocol) servers. MCPHub lets you aggregate multiple MCP servers into separate endpoints with flexible routing strategies — one place to manage all your MCP integrations.

**Tech:** TypeScript · Node.js · MCP Protocol
**Impact:** 2,000+ GitHub stars, 300+ developer community, featured in the official MCP ecosystem
**Links:** [GitHub](https://github.com/samanhappy/mcphub) · [Docs](https://mcphub.io)

### [Selectly](https://github.com/samanhappy/selectly)

An all-in-one AI-powered browser extension for context-aware text processing. Select, translate, explain, summarize, or chat about any text on any webpage — without leaving the page.

**Tech:** TypeScript · Browser Extension APIs · LLM Integration
**Links:** [GitHub](https://github.com/samanhappy/selectly) · [Chrome Web Store](https://chromewebstore.google.com/detail/selectly)

### Dify Plugins

A collection of plugins for the [Dify](https://dify.ai/) platform, extending its document processing capabilities:

- **[Excel Process Plugin](https://github.com/samanhappy/dify-excel-process-plugin)** — Read, parse, and process Excel files within Dify workflows
- **[Word Process Plugin](https://github.com/samanhappy/dify-word-process-plugin)** — Read and extract content from Word documents in Dify workflows

**Tech:** Python · Dify Plugin SDK
