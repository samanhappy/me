## 定义

简单来说，Agent Skills 只是一组 Skill 目录，每个 Skill 目录下包含一组文件：
- SKILL.md：核心定义
- 引用指令
- 可执行代码。

这些文件可以为 agent 提供额外的指令上下文和确定性代码能力。

## 设计原则

渐进式加载是 Agent Skills 的核心设计原则。

## 对比

和 MCP 相比，Skills 更简单，只是文件而已。
