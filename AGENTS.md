# AGENTS GUIDE

## Development
- use `pnpm` as package manager

### pnpm commands
- use `pnpm format` to format code
- use `pnpm format:check` to check code format
- use `pnpm lint` to lint code
- use `pnpm dev` to start dev server
- use `pnpm build` to build the site

## Rules
- run `pnpm format` before commit code
- run `pnpm lint` and `pnpm format:check` in CI
- run `pnpm build` in CI
